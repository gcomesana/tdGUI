In order to the app can be run in your environment, you must update your browser. To do this one:

If you are in a Mac

You shouldn't be reading this as you should have one of the last Safari versions (but the latest!). But, as this is not your case, you have to update to the latest version: run the Software Update pane from the System Preferences and if there is any version suitable for you OS X version, it will give you the latest version available. Follow the instructions to install.

If you are in Windows

You are a bit unlucky regarding to this issues as Safari has no updates for Windows since 5.1.7 version. Check and download and install another browser like Internet Explorer, Chrome or Mozilla Firefox
